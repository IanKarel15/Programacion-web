Introducción

En esta página demostramos diversas etiquetas de HTML para dar forma y significado al contenido. Verás ejemplos de títulos, negritas, cursivas, subrayados, así como etiquetas semánticas como strong y em.

Tip: Puedes usar Ctrl + F para buscar una etiqueta específica.

Formato de texto

Estilos básicos

Negrita (b), Negrita semántica (strong), Cursiva (i), Énfasis semántico (em), Subrayado (u).

También podemos combinar estilos: texto importante y enfatizado o subrayado en negrita.
